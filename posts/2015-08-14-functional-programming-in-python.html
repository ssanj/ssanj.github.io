Higher-Order Functions

In this example the returned m is a reference to the constructor of a class.

def get_class(self, kls):
    parts = kls.split('.')
    module = ".".join(parts[:-1])
    m = __import__( module )
    for comp in parts[1:]:
        m = getattr(m, comp)            
    return m
